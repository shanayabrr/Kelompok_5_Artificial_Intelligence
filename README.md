# 📘 Analisis Perbandingan Model RAG dan Stochastic RAG dalam NLP

## 📖 Deskripsi

Proyek ini merupakan tugas mata kuliah Artificial Intelligence yang membahas analisis perbandingan antara dua model AI, yaitu **Retrieval-Augmented Generation (RAG)** dan **Stochastic RAG** dalam meningkatkan performa sistem Natural Language Processing (NLP).

Penelitian ini bertujuan untuk memahami konsep dasar kedua model, menganalisis kelebihan dan kekurangannya, serta membandingkan performa keduanya dalam berbagai aspek.

---

## 🎯 Tujuan

* Memahami konsep **RAG** dan **Stochastic RAG**
* Menganalisis kelebihan dan kekurangan masing-masing model
* Membandingkan performa kedua model dalam sistem NLP

---

## 🧠 Model yang Dibahas

### 1. Retrieval-Augmented Generation (RAG)

Model yang menggabungkan proses:

* Retrieval (pencarian informasi)
* Generation (pembuatan teks)

**Keunggulan:**

* Meningkatkan akurasi
* Mengurangi hallucination
* Memanfaatkan data eksternal

**Kekurangan:**

* Bergantung pada kualitas data
* Generalisasi terbatas

---

### 2. Stochastic RAG

Pengembangan dari RAG dengan pendekatan probabilistik dan optimasi end-to-end.

**Keunggulan:**

* Performa lebih tinggi
* Lebih optimal dalam pembelajaran

**Kekurangan:**

* Kompleksitas tinggi
* Membutuhkan komputasi besar

---

## 📊 Perbandingan Model

| Aspek        | RAG    | Stochastic RAG |
| ------------ | ------ | -------------- |
| Akurasi      | Tinggi | Lebih tinggi   |
| Kompleksitas | Rendah | Tinggi         |
| Implementasi | Mudah  | Sulit          |
| Penggunaan   | Luas   | Terbatas       |
